This website was created by Tom Davidson as part of a web design project for a fictional company, CF Audio-Visual LLC. The goal of this project was to design and develop a fully functional multi-page website while meeting all required course objectives and technical specifications.

The overall design of the site was based on a custom visual concept. The logo was created using a close-up image of a stereo speaker, enhanced with a textured effect and layered with red and blue color gradients. These colors were then carried throughout the site to maintain a consistent theme, with variations applied to backgrounds, text, and navigation elements to create a cohesive visual experience.


As part of the project requirements, multimedia content was incorporated into the website. This includes a video featured on the Services page, demonstrating a home entertainment setup and showcasing modern audio-visual technology.

Another key requirement was the implementation of a user input form. The Contact page includes a sample form that allows users to select options and enter their information. While the form is not connected to a live backend system, it demonstrates the structure and functionality needed to capture user inquiries in a real-world scenario.

Developing this website provided valuable hands-on experience in HTML and CSS, particularly in building a multi-page site without the use of a graphical editor. This project highlights both technical skills and creative design decisions, and it serves as a foundation for further development as part of ongoing studies in information technology.
